Free macOS speech-to-text app

Transcriptor

Local-first dictation and transcription for macOS, with global voice input, on-device Whisper models, transcript history, and optional cloud providers when you choose them.
Download DMG View Source

Highlights

  • Global voice input shortcut
  • Hold-to-talk and toggle-to-talk modes
  • Local Whisper transcription
  • History, playback, copy, and export

Privacy

  • Local models keep audio on your Mac
  • Cloud upload is opt-in only
  • API keys stay in the macOS Keychain
  • History remains local unless you export

What you can do

  • Import audio files
  • Re-transcribe with another model
  • Insert text into the active app
  • Run from the menu bar

Install

1
Download

Get the latest DMG from GitHub Releases.

2
Drag to Applications

Open the DMG and move Transcriptor into your Applications folder.

3
Grant permissions

Allow microphone access, then optionally grant Accessibility for auto-insert.

4
Download a model

Open Models and install a local Whisper model such as Tiny.

NVIDIA Parakeet and `.webm` import stay visibly unavailable until real native support exists. Transcriptor does not fake working provider or model support.
